# Airflow Plugins Directory

This directory is for custom Airflow plugins, operators, hooks, and macros.

## Directory Structure

```
plugins/
├── operators/
│   └── custom_operators.py
├── hooks/
│   └── custom_hooks.py
├── macros/
│   └── custom_macros.py
└── __init__.py
```

## Examples

### Custom Operator
Place custom operator classes in `operators/custom_operators.py`

### Custom Hook
Place custom hook classes in `hooks/custom_hooks.py`

### Custom Macros
Place custom macro functions in `macros/custom_macros.py`

For more information, visit the [Airflow Plugins Documentation](https://airflow.apache.org/docs/apache-airflow/stable/plugins.html)
